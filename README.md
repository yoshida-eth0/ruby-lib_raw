# LibRaw

LibRaw is a library for reading RAW files obtained from digital photo cameras

## Installation

Add this line to your application's Gemfile:

    gem 'lib_raw'

And then execute:

    $ bundle

Or install it yourself as:

    # brew install libraw
    $ gem install lib_raw
